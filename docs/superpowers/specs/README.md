# Historical tier — specs

Per-milestone specs and brainstorms. **Allowed to go stale.** Each file is a snapshot of the thinking
at the time — *how we got here* — and is **not** the source of truth.

**This path matches the superpowers plugin default** (`superpowers:brainstorming` saves to
`docs/superpowers/specs/YYYY-MM-DD-<topic>-design.md`), so brainstorm output lands here automatically.

- Don't cite these where they conflict with the code or the root `CLAUDE.md`.
- **Migrate before you retire:** before treating a milestone as "done," move any still-current
  decision that lives *only* here into a maintained `docs/*.md` (or `CLAUDE.md` Locked decisions) first.
- **Kept, never deleted.** "Allowed to go stale" refers only to the *content* drifting; the files stay
  forever as your record of *why* decisions were made.

> The founding spec [`../../kickstart.md`](../../kickstart.md) predates this tree and lives at `docs/`
> root (it was shared as a public link — kept in place so the link doesn't break). New specs land here.
